🎮 Exemplo Simples - Emulador NES

📋 Como usar:
1. Carregue um arquivo ROM de NES (.nes) usando o botão abaixo
2. O jogo será carregado automaticamente no emulador
3. Use os controles on-screen ou teclas do teclado para jogar

📁 Carregar ROM do NES

Formatos suportados: .nes, .fds, .unif, .unf

⚠️ Aviso Legal:
Este emulador é apenas para fins educacionais. Certifique-se de possuir legalmente os jogos que você executar. As ROMs devem ser obtidas de seus próprios cartuchos originais.
🎯 Controles do Teclado:
• Setas - Direcionais
• Z - Botão A
• X - Botão B
• Enter - Start
• Space - Select
